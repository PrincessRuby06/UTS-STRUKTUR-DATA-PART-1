## Fitur Program

## Penjelasan
- Bubble Sort : menukar data bersebelahan
- Insertion Sort : mensisipkan ke posisi yang benar
- Selection Sort : pilih nilai yang terbesar

Program menampilkan proses iterasi untuk memperjelas cara kerja algoritma sorting.
